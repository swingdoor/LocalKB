# Change: Replace Bubble Menu Icons

## Summary

Replace hand-written bubble menu SVG icons with semantic `lucide-react` icons while keeping existing menu behavior and layout intact.

## Problem

Several current bubble menu icons do not clearly match their actions:

- Font size looks like image resizing.
- Heading looks like a generic list/menu.
- Font/color controls are hard to distinguish.
- AI polish and expand icons are visually ambiguous.
- Hand-written SVGs have inconsistent visual weight.

## Goals

- Use a consistent icon set for bubble menu controls.
- Improve semantic match between icon and action.
- Keep existing commands, dropdowns, layout, tooltips, and active states.
- Avoid broad visual redesign.

## Non-Goals

- Do not redesign the bubble menu layout.
- Do not change editor commands or document data.
- Do not add new menu actions.
