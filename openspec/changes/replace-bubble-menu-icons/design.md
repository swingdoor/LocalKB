# Design: Replace Bubble Menu Icons

## Decision

Use `lucide-react` for bubble menu action icons.

## Icon Mapping

| Action | Icon |
| --- | --- |
| Font family | `Type` |
| Font size | `ALargeSmall` |
| Heading | `Heading` |
| Font color | `Palette` |
| Bold | `Bold` |
| Italic | `Italic` |
| Strike | `Strikethrough` |
| Link | `Link` |
| Left align | `AlignLeft` |
| Center align | `AlignCenter` |
| Right align | `AlignRight` |
| Justify | `AlignJustify` |
| AI polish | `Sparkles` |
| AI expand | `Expand` |
| Edit image/canvas/mind map | `Pencil` |
| Download | `Download` |
| Confirm/cancel link input | `Check` / `X` |

## Notes

The color button should keep the current color swatch indicator below the icon.
